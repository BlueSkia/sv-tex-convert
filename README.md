# sv-tex-convert

[website](https://tex-convert.eridanus.page)

## What

Web interface to easily convert between FF14's tex format and DirectX's DDS.

## Quick start

```sh
# Install deps
pnpm i

# Dev server (go to http://localhost:5173 by default)
pnpm run dev
```

## Building

```sh
pnpm run build
```
