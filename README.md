# sv-tex-convert

## What

Web interface to easily convert between FF14's tex format and DirectX's DDS.

## Quick start

```sh
# Install deps
pnpm i

# Dev server
pnpm run dev
```

## Building

```sh
pnpm run build
```

Made with [svelte](https://svelte.dev) and [`sv`](https://github.com/sveltejs/cli).
